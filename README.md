# MERN | Chat App - Realtime - Socket.io

- MERN
- Mongo DB
- ExpressJS
- ReactJS
- NodeJS
- Socket.io (Realtime)
- Styled components

## Features/Components
- ReactJS cho UI
- Backend dùng NodeJS
- Authentication dùng JWT
- MongoDB cho database 
- Socket.io để nhắn tin Realtime (thời gian thực)
- CSS - Styled components
- Responsive

## RUN
- (client)
- `cd public`
- `npm install`
- `npm start`

- (server)
- `cd server`
- `npm install`
- `npm start`
